How to run:
Method 1 - 
Go to https://student-performance-prediction-xc.streamlit.app/. 

Method 2 -
1. Open command prompt and go to this folder directory.
2. type: streamlit run app.py

model.ipynb is the file to construct the prediction model.
app.py is the file to run streamlit.
student_data.csv is the dataset file.
